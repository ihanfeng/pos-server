部署外卖项目说明：
一、设置数据库
修改配置文件/resources/dbconfig.properties

二、系统配置1
修改配置文件/resources/sysConfig.properties
1.#solr_url  配置solr地址
2.#alipay   支付宝支付相关配置
3.#wxpay	微信支付相关配置
4.#tenpay   财付通相关配置

三、系统配置2
修改配置文件/com/wm/util/config.xml
1.设置为非测试模式
<config name="isdebug" value="false" />

2.设置项目跟路径(注意:一定要清楚要不要WM)
<config name="domain" value="http://apptest.0085.com/WM" />
